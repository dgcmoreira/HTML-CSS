Praticando

Nesta aula aprendi sobre textos ordenados e não ordenados.

E sobre como podemos formatar os tipos de identificadores de cada item de uma lista.

Ex de uma lista Ordenada:

Rotina da casa:

  1. Ler o examine
    2. Mandar o cmentário no grupo
  2. Leitura da Bíblia
    3. Fazer anotações
  3. Skin care
    4. Certificar dos produtos
  4. Arrumar o quarto
    5. Verificar troca do jogo de cama
  5. Ir trabalhar
    6. Se atentar ao horário

Lista não Ordenada

Desejos:

  • Viajar de avião para Jericoacoara
    1. Criar um cronograma de viagem
    2. Criar o orççamento adequado
    3. Traçar todas a rotas que iremos visitar
  • Comprar uma HR-V
    1. Verificar onde se tem as melhores ofertas
    2. Realizar orçamento plausível
    3. Fechar a compra quando seguro
  • Compra uma cobertura duplex
    1. Encontrar o local adequado
    2. Analizar preços
    3. Assegurar quanto a seriedade do empreendimento
  • Ter uma Menina
    1. Fazer um planejamento futuro
    2. Checar rotinas
    3. Plano pré parto seguro
